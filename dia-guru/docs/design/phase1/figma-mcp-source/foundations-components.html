Accessibility Targets

Contrast

AA minimum for all text/background pairings; warning and error cards validated for readable body text.

Touch Targets

All primary controls sized to 44x44pt minimum. Quick-add exceeds minimum for thumb reach.

Type Scaling

Layout keeps key cards readable at larger text sizes by avoiding fixed-height text containers.

Motion

Only meaningful transitions: sheet in/out, progress updates, and success/error state swaps.
